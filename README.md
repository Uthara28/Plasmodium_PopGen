# Plasmodium_PopGen

This is an IRT3/Master's thesis project vis-à-vis the EES Masters at LMU, Munich Germany. It is being completed at the Population Genetics Lab, under Prof. Dr. Aurélien Tellier and Prof. Dr Dirk Metzler, at the Technical university of Munich, Germany. 


This project explores diversity and structure across chromosomes in Plasmodium sp., with the aim of appliying SMC (sequential markovian chain) models to find multiple merger coalescent events. This will provide key insights into the population genetics of infectious disease evolution and contributes to both an evolutionary and epidemiological understanding of the deadly diseae of malaria.
